Golang-Web编程之模板引擎

2021-04-10 2 分钟阅读 #Golang#Web编程

标准库中的 text/template包是 Go 语言内置的文本模板引擎。

html/template是对text/template的封装，用法大同小异。

常用方法：

 1//创建一个模板对象
 2func New(name string) *Template {...}
 3
 4//从本地文件加载模板
 5func ParseFiles(filenames ...string) (*Template, error) {...}
 6func ParseGlob(pattern string) (*Template, error) {...}
 7
 8//Parse方法接受一个string类型的参数，即文本模板的内容，然后对内容进行解析并返回解析过程中发生的任何错误。
 9func (t *Template) Parse(text string) (*Template, error) {...}
10
11//Execute方法用于渲染模板，该方法接受两个参数：输出对象和指定根对象
12func (t *Template) Execute(wr io.Writer, data interface{}) error {...}
13
14//当加载了多个模板文件时，使用ExecuteTemplate方法指定渲染某一个模板文件
15func (t *Template) ExecuteTemplate(wr io.Writer, name string, data interface{}) error {...}
16
17//除了内置的模板函数外，可以通过Funcs方法增加自定义模板函数
18func (t *Template) Funcs(funcMap FuncMap) *Template {...}

模板中的Action（赋值、逻辑控制、函数等等）需要使用{{}}包裹，除Action之外的内容会原封不动的进行输出。

{{.}}操作符默认指向根对象。

注释的语法和 Go 语言程序代码中的块注释语法相同，即使用 /* 和 */ 将注释内容包括起来，例如：{{/* 这是注释内容 */}}。

{{- 用于剔除模板左侧多余的空格， -}}用于剔除模板中右侧多余的空格

一些示例

直接渲染文本

 1func main() {
 2	http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
 3		temp, _ := template.New("test").Parse("hello template")
 4
 5		_ = temp.Execute(w, nil)
 6	})
 7
 8	log.Println("Starting HTTP server...")
 9	log.Fatal(http.ListenAndServe("localhost:4000", nil))
10}

在模板中引用根对象

 1type Rect struct {
 2	Width int
 3	Height int
 4}
 5
 6func (r *Rect) Area() int {
 7	return r.Width * r.Height
 8}
 9
10func main() {
11	http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
12		temp, _ := template.New("test").Parse(`Rect
13Width: {{.Width}}
14Height: {{.Height}}
15Area: {{.Area}}
16`)
17
18		width, _ := strconv.Atoi(r.URL.Query().Get("w"))
19		height, _ := strconv.Atoi(r.URL.Query().Get("h"))
20
21		rect := &Rect{Width: width, Height: height}
22
23		_ = temp.Execute(w, rect)
24	})
25
26	log.Println("Starting HTTP server...")
27	log.Fatal(http.ListenAndServe("localhost:4000", nil))
28}

在模板中定义变量

1{{$Width := 10}}	//变量名使用$作为前缀，:=赋值
2{{$Height := 20}}
3{{$Height = 30}}	//更改变量值
4Width: {{$Width}}	//获取变量值
5Height: {{$Height}}

剔除空格

1{{- $Width := 10 -}}	//变量名使用$作为前缀，:=赋值
2{{- $Height := 20 -}}
3{{$Height = 30}}	//更改变量值
4Width: {{$Width}}	//获取变量值
5Height: {{$Height}}

给模板添加自定义函数

 1func main() {
 2	http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
 3		temp := template.New("test").Funcs(template.FuncMap{
 4			"area": func(a, b int) int {
 5				return a * b
 6			},
 7		})
 8
 9		_, _ = temp.Parse(`
10Area: {{area 1 2}}
11`)
12
13		temp.Execute(w, nil)
14	})
15
16	log.Println("Starting HTTP server...")
17	log.Fatal(http.ListenAndServe("localhost:4000", nil))
18}
