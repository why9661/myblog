Golang-方法

2021-04-28 1 分钟阅读 #Golang#Golang

方法本质上是一个函数，方法接收者就是隐含的第一个参数

 1type A struct {
 2	Name string
 3}
 4
 5func (a A) SayHello1() string {
 6	return "Hello " + a.Name
 7}
 8
 9func SayHello2(a A) string {
10	return "Hello " + a.Name
11}
12
13func main() {
14	t1 := reflect.TypeOf(A.SayHello1)
15	t2 := reflect.TypeOf(SayHello2)
16	fmt.Println(t1 == t2)  // true
17}

以下是一个常见的方法调用，这其实是一个语法糖，等价于注释行的函数调用。

 1type A struct {
 2	Name string
 3}
 4
 5func (a A) SayHello() string {
 6	return "Hello " + a.Name
 7}
 8
 9func main() {
10	a := A{"why"}
11	fmt.Println(a.SayHello())
12	//fmt.Println(A.SayHello(a))
13}

如果需要通过方法调用来完成对接收者的修改，那么需要使用指针接受者

 1type A struct {
 2	Name string
 3}
 4
 5func (a A) Rename1(name string) {
 6	a.Name = name
 7}
 8
 9func (a *A) Rename2(name string) {
10	a.Name = name
11}
12
13func main() {
14	a := A{"why"}
15	b := A{"why"}
16	a.Rename1("why123")
17	b.Rename2("why123")
18	fmt.Println(a.Name) //why
19	fmt.Println(b.Name) //why123
20}

可以通过值调用指针接受者的方法，也可以通过指针调用值接收者的方法，这也是一种语法糖

 1type A struct {
 2   Name string
 3}
 4
 5func (a A) Rename1(name string) {
 6   a.Name = name
 7}
 8
 9func (a *A) Rename2(name string) {
10   a.Name = name
11}
12
13func main() {
14   a := A{"why"}
15   b := &A{"why"}
16   a.Rename2("why123") //在编译阶段会转换为(&a).Rename2(string)
17   b.Rename1("why123") //在编译阶段会转换为(*b).Rename1(string)
18   fmt.Println(a.Name) // why123
19   fmt.Println(b.Name) // why
20}

方法和函数一样可以赋值给变量

 1type A struct {
 2   Name string
 3}
 4
 5func (a A) PrintName() {
 6   fmt.Println(a.Name)
 7}
 8
 9func main() {
10   a := A{"why"}
11   f1 := a.PrintName //有捕获列表的Function Value
12   f2 := A.PrintName //Function Value
13   f1()   //why
14   f2(a)  //why
15}
