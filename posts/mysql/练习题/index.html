  • 查询"01"课程比"02"课程成绩高的学生的信息及课程分数；

    解析：内连接join和别名as的使用

    1select a.*,t3.c1_score,t3.c2_score from
2(select t1.sid,t1.c1_score,t2.c2_score from
3(select sid,score as c1_score from Score where cid='01') as t1
4join
5(select sid,score as c2_score from Score where cid='02') as t2
6on t1.sid=t2.sid where t1.c1_score>t2.c2_score) as t3
7join
8Student as a on a.sid=t3.sid;

  • 查询学生选课存在" 01 “课程但可能不存在” 02 “课程的情况（不存在时显示为 null);

    解析：左连接left join的使用

    1select * from
2(select * from Score where cid='01') as t1
3left join
4(select * from Score where cid='02') as t2
5on t1.sid=t2.sid;

  • 查询平均成绩大于等于 60 分的同学的学生编号和学生姓名和平均成绩；

    解析：分组查询group by…having…以及聚合函数avg的使用

    1select t2.sid,t2.sname,t1.avg_score from
2(select sid,avg(score) as avg_score from Score group by sid having avg(score)>=60) as t1
3join
4Student as t2 on t1.sid=t2.sid;

  • 查询在 Score 表存在成绩的学生信息；

    1select * from Student where sid in (select distinct sid from Score);

  • 查询「李」姓老师的数量；

    解析：count函数like关键字以及通配符%的使用。

    1select count(*) from teacher where tname like '李%';

  • 查询各科成绩最高分、最低分和平均分,以如下形式显示：

    课程 id，最高分，最低分，平均分，及格率，中等率，优良率，优秀率

    及格为>=60，中等为：[70,80)，优良为：[80-90)，优秀为：>=90

    解析：max、min、avg、sum函数case when语句以及分组group by的使用

    1select cid as 课程iD,
2max(score) as 最高分,
3min(score) as 最低分,
4avg(score) as 平均分,
5sum(case when score>=60 then 1 else 0 end)/count(sid) as 及格率,
6sum(case when score>=70 and score<80 then 1 else 0 end)/count(sid) as 中等率,
7sum(case when score>=80 and score<90 then 1 else 0 end)/count(sid) as 优良率,
8sum(case when score>=90 then 1 else 0 end)/count(sid) as 优秀率
9from score group by cid;

  • 按各科成绩进行排序，并显示排名;

    解析：排序函数rank() over()、dense_rank() over()、row_number() over()的区别和使用,以及分区partition by的使用

    1select *, rank() over(partition by cid order by score desc) as ranked from score;

    1select *, dense_rank() over(partition by cid order by score desc) as ranked from score;

    1select *, row_number() over(partition by cid order by score desc) as ranked from score;

  • 查询学生的总成绩，并显示排名;

    1select *, (rank() over(order by t1.total desc)) as ranked from
2(select sid,sum(score) as total from score group by sid) as t1;

  • 查询出只选修两门课程的学生学号和姓名;

    1select sid,sname from student
2where sid in (select sid from score group by sid having count(cid)=2);

  • 查询 1990 年出生的学生名单;

    解析：year函数的使用

    1select * from student where year(sage)=1990;

  • 查询各学生的年龄，只按年份来算；

    1select sname,(year(now())-year(sage)) as age from student;

  • 按照出生日期来算，当前月日 < 出生年月的月日则，年龄减一；

    解析：date_format函数的使用

    1select sname,
2(case when (date_format(now(),'%m-%d')-date_format(sage,'%m-%d')) < 0
3 then year(now())-year(sage)-1
4 else year(now())-year(sage)
5 end) as age 
6from student;

    解析：week函数的使用（week函数返回的是今年的第几周）

    1select * from student where week(now())=week(sage);
